# Dataset

This directory contains public samples from the dataset used to fine-tune LIMBA.

The training data follows an instruction–context–response structure and includes examples covering:

- Italian–Limba Sarda Comuna translation
- Conversational interaction
- Sardinian language and culture
- STEM and technology
- General knowledge
- Model identity and assistant behaviour

The complete training dataset is not currently distributed in this repository. The included sample is provided to document the dataset structure and the range of tasks addressed during fine-tuning.

# Example Prompts

The following prompts demonstrate some of the tasks currently supported by LIMBA.

## Italian → Limba Sarda Comuna translation

```text
Traduci in sardo: L'algoritmo di machine learning sta analizzando i dati.
```

## Educational explanation

```text
Ispiegami su teorema de Pitagora.
```

## Programming assistance

```text
Iscrie unu script in Python pro lèghere unu file CSV.
```

## General conversation

```text
Spiegami che cos'è l'intelligenza artificiale.
```

## Sardinian text generation

```text
Scrivi un breve testo in Limba Sarda Comuna sulla tutela dell'ambiente.
```

## Suggested testing criteria

When testing LIMBA, evaluate:

- linguistic consistency;
- translation fidelity;
- instruction following;
- factual accuracy;
- clarity and completeness;
- stability across repeated prompts.
